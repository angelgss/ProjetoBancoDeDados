@mui/icons-material inclui os mais de 2.100 ícones oficiais do Material convertidos em SvgIconcomponentes. Depende de @mui/material, que requer pacotes Emotion. Use um dos seguintes comandos para instalá-lo:

npm install @mui/icons-material @mui/material @emotion/styled @emotion/react

Consulte a página Instalação para obter documentos adicionais sobre como garantir que tudo esteja configurado corretamente.

O Google também oferece o Material Symbols como sucessor do Material Icons. @mui/icons-materialNo momento, ele abrange apenas ícones. Ainda não há suporte para símbolos.
